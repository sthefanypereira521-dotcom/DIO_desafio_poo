 # Mini Sistema Bancario AWS


## estrutura

1. Cliente envia requisicao.
2. API Gateway recebe a requisicao.
3. IAM controla permissoes de acesso.
4. EC2 executa a aplicacao FastAPI.
5. Lambda processa regras de negocio.
6. SQLite3 armazena os dados.
7. S3 mantem backups do banco.

## Tecnologias

- AWS API Gateway
- AWS IAM
- AWS EC2
- AWS Lambda
- AWS S3
- SQLite3
- FastAPI
- Python

## Objetivo

Projeto  para demonstrar uma arquitetura simples de sistema bancario utilizando servicos AWS e FastAPI.
